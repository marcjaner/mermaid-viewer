# Ubiquitous Language

| Term | Definition | Context | Example |
| --- | --- | --- | --- |
| Diagram Source | Raw Mermaid text used as the single source of diagram intent. | CLI, Viewer, Rendering | `flowchart TD\\nA-->B` |
| Render Job | One deterministic conversion of Diagram Source + Style Profile into an artifact. | Rendering | `mmdv render --format png --output diagram.png` |
| Style Profile | Portable set of theme, themeVariables, and CSS options. | Config, Viewer, CLI | `theme: dark` plus `themeVariables` |
| Share URL | URL with compressed hash payload and optional style query param. | Viewer, CLI | `...?style=<payload>#<diagramHash>` |
| Diagram Hash | Deflate+base64 payload stored in URL hash. | Codec | `#eJx...` |
| Artifact | File output generated by rendering. | CLI | `diagram.svg`, `diagram.png` |
| Machine Result | Versioned JSON object returned by CLI in `--json` mode. | CLI | `{"schemaVersion":"1.0","ok":true,...}` |
| Codec | Contract to encode/decode Diagram Source in URL-safe form. | Shared | `encodeHashPayload`, `decodeHashPayload` |
